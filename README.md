# Лабораторная работа 2 — Настройка рабочего окружения для веб-разработки

Проект создан в рамках лабораторной работы по ИТиВП.

## Используемые технологии

- HTML5, CSS3, JavaScript
- Node.js, npm
- Git
- VS Code, расширения: Live Server, Auto Rename Tag, Prettier, GitLens, ES6 String HTML, Figma to Code

## Структура проекта

- `index.html` — главная страница
- `css/style.css` — базовые стили
- `js/script.js` — скрипты
- `assets/`, `images/`, `design/` — ресурсы и макеты
